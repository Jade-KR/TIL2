# 리눅스

디렉토리

- / - 루트 디렉토리
- Home - 사용자별 디렉토리
- Media, mnt - 장치를 연결해 쓸 수 있는 디렉토리
- usr - 프로그램 설치
- etc - 프로그램을 실행하기위한 설정들
- var - 설정을 위해 만들어지는 값들
- bin - 모든 유저가 사용하는 실행파일
- sbin - admin 유저가 사용하는 실행파일
- tmp - 임시로 데이터를 풀었다 삭제를 할때 사용하는 디렉토리
- sys - 시스템에 대한 설정들



### 파일 경로와 순회하기

- pwd
- ls
- cd
- man ls - 도움말



### 파일 시스템을 위한 명령어

- Mkdir - 디렉토리 생성
- Rmdir - 디렉토리 삭제
- touch - 빈 파일 생성
- mv - 파일 이동 / 변경
- rm - 파일 삭제
- cp - 파일 복사
- Rm -r 디렉토리/ - 디렉토리 안에 모든 것들 다 지우기
